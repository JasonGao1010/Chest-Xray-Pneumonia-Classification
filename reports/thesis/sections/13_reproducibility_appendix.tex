\chapter{复现与验收附录}

\section{环境安装}
\begin{verbatim}
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m pytest -q
\end{verbatim}

\section{患者级划分与审计}
\begin{verbatim}
python scripts/prepare_kermany_grouped.py
python scripts/audit_integrity.py \
  --dataset data/processed/kermany_grouped_seed42 \
  --output results/integrity_audit_grouped.json
\end{verbatim}

正式manifest为\resultfile{data/splits/kermany\_grouped\_seed42.csv}。重新生成前必须确认不会覆盖已用于正式结果的冻结版本。

\section{严格训练示例}
\begin{verbatim}
python scripts/train.py \
  --config configs/densenet121.yaml \
  --data-root data/processed/kermany_grouped_seed42 \
  --seed 42 \
  --json-output results/strict_train_densenet121_seed42.json
\end{verbatim}

评价命令从training summary读取best checkpoint，再分别评价Kermany locked test与RSNA test。所有正式命令显式指定输出名并使用\texttt{--no-update-latest}，避免通用latest文件覆盖证据。

    results/strict_predictions_rsna_val_\
